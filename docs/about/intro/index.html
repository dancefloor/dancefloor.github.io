Introduction

This website works as the main documentation of the Dancefloor Management System (DMS).

#Mission

DMS is an IT solution created to help dance schools run their business successfully by focusing on three main aspects:

  • The automatization of common administratives, organizational and marketing tasks that will largely save time to teachers and managers
  • A set of programs with dancing lessons, methodologies and guides that will help teachers during their lessons improving the relationship between teachers and students
  • A dashboard with relavant KPIs that will help teachers make the best decisions on strategies and plans
