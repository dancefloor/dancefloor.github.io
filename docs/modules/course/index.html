Courses Module

The course module manages all the course features:

#Features

  • Create, Read, Update, Delete a course
  • Only an admin, a teacher or an assistant can create, update or delete a course
  • An active registered user has the priority for the following registration session
  • If teachers field left empty, it means dancefloor team
  • Frontend: Grid view list of courses with options to register
  • Frontend: Detail view of course with register or sign-up button for user according to role
  • Frontend: Number of available places
  • Frontend: Number of available places for a women
  • Frontend: Number of available places for a men
  • Frontend: Display the prices according to the user's working status and options (duo, promo)
  • Backend: Table view list of courses with options to edit, and delete
  • Backend: Display calendar graph of assistance per lesson
  • Backend: Display: Benefit / break even point
  • Backend: Display Line chart evolution of class assistance, with capacity and break even point lines
  • Backend: Students that have not paid this course yet
  • Backend: Statistics gender and group age
  • Backend: Average of assistance per session

#Relationships

  • A course takes place in one and only one location
  • A course is giving by one or many teachers|assistants
  • A course can have one or more tags
  • A user can register to one or more courses
  • A course can belong to one and only one style
  • A course has one or more lessons (List of presence)
  • A course has one previous class (Registration priority for student)
  • A course has one following class (Registration priority for student)
