Registration Module

#Features

  • Verify available places (location capacity)
  • Verify available places by gender
  • Parity assessment (equal amount of males and females)
  • Indicator of available places (display number of places for a given course)
  • Backend Display indicator class proportion (X men/ X woman)
  • The registration automatically adjust the price according to
  • The user can select duo promo or special promo
  • Payment reminders after a specific time (to be define)

#Relationships

  • A user can register to a course
  • A registration involes one and only one payment (belongsToPayment)

#Pending

This status signifies that the used has register but he has not paid (place not garanteed)

#Waiting

This status signifies that the user attempted to registered but is on a waiting listing

#Registered

This status signifies that the used has register he has paid the course successfully (place garanteed)

#Cancelled

This status signifies tha the user has cancelled his or her registration
