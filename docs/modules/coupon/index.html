During the payment process if the user inputs a coupon code, the system will check if the coupon is valid according to its type:

  • Checks if the coupon has not expire
  • The system will always check if the coupon(coupon id & number) has never been used by the same user (user id)

#Time promos

  • Specif promo in time

#For Groups

  • check if the coupon valid for a specific group of people (style of dance, or specific class)

#Specific person

  • If the coupon is for a specific person, the system will verify during the payment process that the payment corresponds to the correct person. If OK, then will verify that person didn't use this code before
