User Module

This module manages all the features related to a user. A user is the representation of a person, its use cases, abilities, options. Here is the list of all the features of the user module:

  • A user can create an user account (Sign up)
  • A user can login into their account (Profile page)
  • A user can reset password if this has been forgotten
  • A user can view a list courses and a course detail
  • A user can register for a courses
  • A user have roles (default: student)
  • A user can like/dislike a course
  • A user can view and make payments for registered courses
  • A user can favorite a video
  • A user:admin|teacher|assistant can create|udate|delete a course
  • A user:admin|teacher|assistant can create|udate|delete a location
  • A user:admin|teacher|assistant can create|udate|delete a skill
  • A user:admin|teacher|assistant can create|udate|delete a program
  • A user:admin|teacher|assistant can create|udate|delete a style
  • A user:admin|teacher|assistant can create|udate|delete a lesson
  • The list of users can be filtered by course|style group|role|gender|payment status|level
