Location Management

This module manages classrooms

#Features

  • Admin can CRUD locations
  • Frontend: View locations details in Course description
  • Backend: Display popularity
  • Backend: Display rentability
  • Display map

#Relationships

  • A course can take place in one and only one place
  • A location has one ore more pictures
  • A user can like/dislike a location
